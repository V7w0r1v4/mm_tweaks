INTRODUCTION
------------
The version of Splitter.js bundled with Monster Menus has been abandoned since
January 2009.  It has compatibility issues with IE below 11 and jQuery 1.5+.
This module will replace the bundled Splitter.js with an updated fork:
https://github.com/thejameskyle/splitter


INSTALLATION
------------
Enable this module.
