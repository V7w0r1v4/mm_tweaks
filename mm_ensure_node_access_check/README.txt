INTRODUCTION
------------
Monster Menus Ensure Node Access Check alters all "node/foo/bar" menu items to
guarantee that MM permissions are enforced in addition to running the menu
item's normal access callback.


INSTALLATION
------------
Install similar to other contributed Drupal modules.

See https://www.drupal.org/documentation/install/modules-themes/modules-7 for
more information.


CONFIGURATION
-------------
The module works out of the box with Monster Menus.  No further configuration is
necessary.
